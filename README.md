# 📧 **BASE APPAREL**

**Landing page de inscrição para newsletter**  
Um site moderno que oferece uma interface limpa e responsiva para **captura de e-mails**, com foco em **design minimalista, tipografia e experiência do usuário**.

🔗 **Demo online:** [baseapparel-cs.vercel.app](https://baseapparel-cs.vercel.app/)

## 🚀 **Tecnologias Utilizadas**

Este projeto foi desenvolvido com:

- **HTML5** – marcação semântica e acessível  
- **CSS3** – Flexbox, Grid, animações e transições  
- **JavaScript (ES6+)** – validação de e-mail no formulário

---

## 📚 **Aprendizados**

Durante o desenvolvimento deste projeto, pratiquei:

- 💻 Criar um **layout responsivo** e fiel ao design  
- 🎨 Escrever **CSS modular e organizado**  
- 📝 Implementar **validação de e-mail** em JavaScript  
- 🖌️ Aplicar conceitos de design: **tipografia, cores e hierarquia visual**

---

## ⚙️ **Funcionalidades**

- ✅ Layout responsivo para desktop e mobile  
- ✅ Validação de e-mail no formulário  
- ✅ Estrutura de CSS modular e reutilizável  
- ✅ Foco em UX e design minimalista
